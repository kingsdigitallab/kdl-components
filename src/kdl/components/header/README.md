# Header

## Data

| Name                | Type   | Description                                 |
| ------------------- | ------ | ------------------------------------------- |
| title               | string | The header title.                           |
| url                 | string | URL to link the header to, defaults to `/`. |
| feature.image       | string | Path to the feature image.                  |
| feature.description | string | Feature image description.                  |
