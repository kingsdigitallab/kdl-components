# Link

## Data

| Name     | Type   | Description                                                      |
| -------- | ------ | ---------------------------------------------------------------- |
| title    | string | Link content. If the `html` option is set the title is not used. |
| page.url | string | URL to link to.                                                  |

## Options

| Name | Type   | Description                        |
| ---- | ------ | ---------------------------------- |
| html | string | HTML to be placed inside the link. |
